## About Arcane Ledger

Arcane Ledger is an independent mobile application that helps players track life totals, general damage, poison counters, game tokens, and the full game log during play. It is designed to be readable across the table, fast to operate, and detailed enough to settle any dispute — without requiring an account, an internet connection, or any personal information from its users.

### Key Features

- Life tracking for 2–6 players with color-coded health indicators
- General damage tracking with Partner General support (dual P1/P2 drums)
- Scroll-wheel drums for Poison, Energy, Experience, and Deck counters
- Monarch, Initiative, and Ascension token assignment
- Judge-grade game log with source registration for every life change
- Post-game summary with life arc sparklines per player
- Full match log export
- Works completely offline — no account, no data collection, no ads

---

## Privacy at a Glance

Arcane Ledger **collects no personal data of any kind.**

- No account required
- No network requests during gameplay
- All data (player names, game history) stays on your device only
- No analytics, no advertising SDKs, no third-party tracking

The only external request the app makes is loading Google Fonts on first launch, which is a standard font file download and not specific to any user.
